# Validate on Blur

If you pass `validateOnBlur={true}` to a field, that field will then be validated every time
the blur event happens on the field.

```tsx
import {Form, Text} from "../src";

// ensure the field has over 5 chars
const validateLength = (val: string) => val && val.length > 5;

<Text field="username" validate={validateLength} validateOnBlur={true}/>
```

Try clicking into this field, and then click away.

# rx-form ![](https://github.com/WeareJH/rx-form/workflows/Node%20CI/badge.svg)

A new form library for Preact + React. 

```tsx
import { Form, Text } from '@wearejh/rx-form';

const SignIn = () => {
    return (
        <Form onSubmit={(values) => console.log(values)}>
            <label>
                Username:
                <Text field="username" />
            </label>
            <label>
                Password:
                <Text field="password" />
            </label>
            <button type="submit">Submit</button>
        </Form>
    )
}
```
